# Eco Bright – E-Waste Management System

## Overview

Eco Bright is a web-based E-Waste Management System designed to promote responsible electronic waste disposal through smart bin tracking and location-based services. The platform helps users locate nearby e-waste collection bins, monitor bin fill levels, and contribute to a cleaner and more sustainable environment.

## Features

### User Authentication

* User Registration and Login
* Secure account management
* Personalized user access

### Smart E-Waste Bin Locator

* Interactive map interface using geolocation
* Display nearby e-waste collection bins
* Real-time bin information and locations
* Quick access to collection centers

### Fill Level Monitoring

* Track current fill levels of e-waste bins
* Visual fill indicators
* Capacity monitoring for efficient waste collection
* Prevent overflow situations

### Route Navigation

* Get directions to selected e-waste bins
* Location-based navigation support
* Easy access to disposal centers

### Bin Management

* Update bin fill levels
* Manage collection point information
* Dynamic database updates

### Database Integration

* MongoDB-based data storage
* Bin location management
* User information storage
* Collection and tracking records

## Workflow

1. User registers or logs into the platform.
2. The system displays nearby e-waste bins on an interactive map.
3. Users can view:

   * Bin location
   * Address details
   * Current fill level
4. Users can obtain directions to the nearest collection bin.
5. Bin fill levels can be updated for monitoring and management.
6. Data is stored and managed through MongoDB.
